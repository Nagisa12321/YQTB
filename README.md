|       名称        | 内容                                                         |
| :---------------: | ------------------------------------------------------------ |

|  `MAIL_PASSWORD`  | 可选，163邮箱授权码                                          |
| `PUSH_PLUS_TOKEN` | 可选，微信扫码登录后一对一推送或一对多推送下面的token，详细见[官网](https://www.pushplus.plus/) |
